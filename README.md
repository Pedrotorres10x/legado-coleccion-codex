# Casa Wow Catalog

Buyer-side real estate catalogue for the Costa Blanca, built to help international buyers discover, compare and move forward with the right properties.

## Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Supabase

## Local development

```sh
npm install
npm run dev
```

## Production build

```sh
npm run build
```
